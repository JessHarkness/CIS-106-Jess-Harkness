Program description--
This program takes data from the Plant Catalog and displays each plant's common name, botanical name,
zone, light, and price. The program will display an item count and the average price of the items in the
catalog.

Main program
    assign file name to a constant
    if file exists:
        process common names list
        process botanical names list
        process zones list
        process lights list
        process prices list
        display plant items
        calculate average price
        display statitiscs
    else:
        print "File does not exist"
        
Process common names list
    Parameters: 
        plant catalog
    Process:
        define empty common names list
        open plant catalog:
            for lines in file:
                strip uneeded characters
                append common names list with common names
    Return:
        common names list
        
Process botanical names list
    Parameters:
        plant catalog
    Process:
        define empty botanical names list
        for lines in file:
            strip uneeded characters
            append botanical names list with botanical name
    Return:
        botanical names list
        
Process zones list
    Parameters:
        plant catalog
    Process:
        define empty zones list
        for lines in file:
            strip uneeded characters
            append zones list with zone
    Return:
        zones list
        
Process light list
    Parameters:
        plant catalog
    Process:
        define empty lights list
        for lines in file:
            strip uneeded characters
            append lights list with light
        Return:
            lights list
            
Process prices list
    Parameters:
        plant catalog
    Process:
        define empty prices list
        for lines in file:
            strip uneeded characters
            append prices list with price
    Return:
        prices list
        
Display plant items
    Parameters:
        common names list
        botanical names list
        zones list
        lights list
        prices list
    Process:
        display plant items as: common name (botanical name) - zone - light - price
    Return:
        None
        
Calculate average price
    Parameters:
        prices list
    Process:
        calculate average: sum of items in list / length of list
        round average to 2 decimal places
    Return:
        average price
        
Display statistics
    Parameters:
        average price
        prices list
    Process:
        display length of average price list (this will be the number of items in the catalog)
        display average price
    Return:
        None
