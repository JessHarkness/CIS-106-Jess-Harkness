# This program takes a file containing names and test scores,
# isolates the scores, and calculates the minimum, maximum,
# and average score.

Main program
    Define names and scores list
    Process list
    Display list
    Calculate max score
    Calculate min score
    Calculate average score
    Display min, max, average

Define names and scores list
    Parameters:
        scores.txt file
    Process:
        verify file
        open file
        assign scores list to file  
    Return:
        names and scores list
        
Process scores list
    Parameters:
        names and scores list
    Process:
        process list to remove names and only retain scores
    Return:
        scores list
        
Display list
    Parameters:
        scores list
    Process:
        display list
    Return:
        none

Calculate max score
    Parameters:
        scores list
    Process:
        calculate max score from list
    Return:
        maximum
    
Calculate min score
    Parameters:
        scores list
    Process:
        calculate min score from list
    Return:
        minimum

Calculate average score
    Parameters:
        scores list
    Process:
        calculate average (sum / length of list)
    Return:
        average

Display min,max,average
    Parameters:
        minimum
        maximum
        average
    Process:
        Display min, max, average
    Return:
        None
